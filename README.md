## Coursera-Getting-and-Cleaning-Data-Course-Project

This repository includes my scripts and documentation for Coursera - Data Science Track / Getting and Cleaning Data Project 

The data used in this project is Human Activity Recognition Using Smartphones research data, which can be observed from the following link 

http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones


Description of the Files Shared: 

Codebook.md includes variables names and the all transformations done during data preparation proccess 

run_analysis.R includes script used for data manipulation 

finally avg.txt includes results
